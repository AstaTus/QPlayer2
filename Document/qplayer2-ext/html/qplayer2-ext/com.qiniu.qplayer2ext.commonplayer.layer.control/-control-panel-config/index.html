播放面板配置

Parameters

name

面板类型（唯一ID）

elements

针对不同的屏幕类型(半屏，横屏全屏，竖屏全屏),对应不同layout 和视频方向

比如点播视频是有seekbar 起播暂停按钮，但是直播没有seekbar 起播暂停按钮， 那么他们的layout资源应该是2个 再比如横屏视频全屏以后 会旋转activity，而竖屏的视频全屏以后不会旋转activity 只会上下充满屏幕，根据业务的需求，一半情况下 他们的layout 也应该是2个
